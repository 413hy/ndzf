# Fzdn1 TG Tool 员工使用说明

## 只双击这个文件

请只双击：

```text
Fzdn1_TG_Tool_v4.1.exe
```

不要双击：

```text
Fzdn1_TG_Tool_v4.1.real.exe
```

`real.exe` 是内部主程序，直接运行它会绕过启动器，可能出现“无法连接服务器”。

## 第一次使用

第一次在本机运行时，如果没有授权文件，软件会进入授权页面。

在授权页面输入公司给的卡密或任意公司内部卡密，然后点击激活。成功后会在本机生成：

```text
license.dat
```

以后再双击 `Fzdn1_TG_Tool_v4.1.exe`，如果 `license.dat` 存在并有效，会直接进入首页。

授权有效期：

```text
2199-12-31
```

## 重复启动限制

现在软件只能运行一次：

- 程序没有运行时，会正常打开。
- 程序已经运行时，再次双击会提示“程序正在运行中，请勿重复启动。”

## 不能删除的文件

- `Fzdn1_TG_Tool_v4.1.exe`
- `Fzdn1_TG_Tool_v4.1.real.exe`
- `config.json`
- `sessions` 文件夹
- `session_config.json`
- `session_string_cache.json`
- `ui`、`webview`、DLL、pyd 等依赖文件

不要只复制一个 exe 给同事，必须复制整个 `fdzn` 文件夹。

## 换电脑或重装系统

授权文件绑定本机。换电脑、重装系统、删除 `license.dat` 后，重新双击 `Fzdn1_TG_Tool_v4.1.exe`，在授权页面再次输入卡密激活即可。
